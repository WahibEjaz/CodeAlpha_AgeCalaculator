# Age Calculator

## Description
This is a simple web application that calculates a user's age based on their date of birth.

## Technologies Used
- HTML
- CSS
- JavaScript

## Features
- Enter date of birth
- Calculates age in years, months, and days
- Simple and user-friendly interface

## How to Run
1. Download or clone the repository
2. Open index.html in your browser
